5 Navigation Primitives

  • • [P] Persistent: name → inode → block
  • • [M] Memory: PID → EPROCESS → VA → PA
  • • [L] Log: timestamp → record → field
  • • [Q] Query: endpoint + cursor → result_set
  • • [C] CAS: hash → blob → content graph
